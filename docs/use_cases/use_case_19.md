# UC-19 – Visualizar Alertas Clínicos

## Atores
- Veterinário
- Estagiário
- Administrador

## Descrição
Permite visualizar alertas automáticos gerados pelo sistema referentes a prescrições atrasadas ou próximas do horário.

## Pré-condições
- Usuário autenticado
- Existência de alertas ativos

## Fluxo Principal
1. Sistema identifica aplicações atrasadas ou próximas do horário programado
2. Sistema exibe alerta visual no painel do usuário
3. Usuário pode acessar o internamento correspondente

## Pós-condições
- Nenhuma alteração automática é realizada
